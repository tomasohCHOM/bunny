# Bunny Detour

A basic 3D website created with Threlte, SvelteKit, Three.js and TypeScript.

## Setup / Development

```bash
# clone this repository
git clone https://github.com/tomasohCHOM/bunny.git

# install dependencies
npm install

# Serve the application in localhost:5173
npm run dev
```
